BOX MODEL: modelo de caixa
Cada elemento recebem o padrão de uma caixa

Exibição do tipo block e do tipo inline: display
Block ocupa linha inteira
Inline ocupa somente o tamanho da caixa; em linha

Display: flex - coloca os elementos de dentro da caixa ao lado do outro (por fora é block)
e aplica nos filhos

largura: height
altura:width
espaçamentos:
preenchimentos interno da caixa: padding
bordas: border
alinhamentos: align-items //flexbox 
justificar conteudo: justify-content
posicionamento:
margem: margin //tecnica pra deixar conteudo de uma box block ao meio usando auto
margin: x y; x: top e bottom y: left e right
x y z; x: top, y: left right, z: bottom
